# SevaSetu v2 – Smart Volunteer & Help Matching Platform

## Overview

SevaSetu is a web-based platform designed to connect individuals in need with volunteers in an efficient and user-friendly manner. It streamlines the process of posting requests and enabling others to respond, creating a structured and accessible support system.

## My Contributions

This repository represents my independently developed version of the SevaSetu project, focusing on improving usability, functionality, and overall user experience.

Key enhancements include:

* Redesigned and improved UI using React for better usability
* Refined authentication flow for smoother user interaction
* Fixed issues in request handling and data flow
* Enhanced navigation and overall user experience
* Deployed the application using Firebase for real-world accessibility

## Features

* User authentication (Login / Registration)
* Create and manage help requests
* Browse and respond to requests
* User profile management
* Responsive and clean user interface

## Tech Stack

* Frontend: React.js
* Backend: Firebase
* Styling: CSS
* Deployment: Firebase Hosting
