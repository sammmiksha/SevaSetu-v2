# SevaSetu v2 – Smart Volunteer & Help Matching Platform

## Overview

SevaSetu is a web-based platform designed to connect people in need with volunteers efficiently. It acts as a bridge between requesters and helpers.

## My Contributions

This repository represents my independent version and improvements over the original project.

* Redesigned UI using React
* Improved authentication flow
* Fixed bugs in request handling system
* Improved user experience and navigation
* Deployment using Firebase

## Features

* User authentication (Login/Register)
* Post help requests
* View and respond to requests
* Profile management
* Clean and responsive UI

## Tech Stack

* Frontend: React.js
* Backend: Firebase
* Styling: CSS
* Deployment: Firebase Hosting
